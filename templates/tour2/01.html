Task A: Dauka and Bauyr

Dauka and Bauyr are very greedy kids. They collect letters of the English alphabet and cannot share the next letter. To decide who will get the letter, they use their names "daukah" and "bauyr." If the letter appears in Dauka's name before it appears in Bauyr's name, it belongs to Dauka, and vice versa. If the letter appears in both names at the same position, it belongs to both. If the letter is not present in both names, then it belongs to no one.

Input Format

A single line containing a lowercase English alphabet letter.

Output Format

Output "daukah" if Dauka gets the letter, "bauyr" if Bauyr gets the letter, "both" if both get the letter, and "none" if no one gets the letter.

Examples

Input:
a
Output:
both

Input:
b
Output:
bauyr

Input:
c
Output:
none

Input:
d
Output:
daukah

Solution

Author: @Mazgagzam


a = input()
s1 = "daukah"
s2 = "bauyr"

if a in s1 and a in s2:
    if s1.index(a) < s2.index(a): 
        print(s1)
    elif s1.index(a) == s2.index(a):
       print("both")
    else:
       print(s2)
elif a not in s2 and a in s1:
    print(s1)
elif a not in s1 and a in s2:
    print(s2)
else:
    print("none")
