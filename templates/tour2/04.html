Problem D: Pair of Numbers

Given an array a consisting of n integers, check if there exist two distinct indices i and j such that ai · aj is equal to the sum of all other numbers in the array.

Input Format

In the first line, the length of the array n is given (3 ≤ n ≤ 105). In the second line, the array itself is given (1 ≤ ai ≤ 105).

Output Format

If there are two elements in the array such that their product is equal to the sum of all other elements, print "YES". Otherwise, print "NO".

Examples

Input:
5
5 6 3 2 7
Output:
YES

Input:
4
1 2 3 6
Output:
NO

Solution

Author: @wilbursoot_fan


n = int(input())
a = sorted([*map(int, input().split())])
s = sum(a)
l = 0
r = n - 1

while l < r:
    if a[l] * a[r] < s - a[l] - a[r]:
        l += 1
    elif  a[l] * a[r] > s - a[l] - a[r]:
        r -= 1
    else:
       print("YES")
       exit()
print("NO")

Is it efficient enough?

Very efficient, passes the tests easily

Complexity:

Time complexity: O(n log n)

Explanation:

Firstly, we sort the array from the smallest to the largest (complexity: n log n). Since our array is sorted, we can apply the two-pointer method. We create two pointers - left and right. In the case where the product ai * aj is greater than the sum of the array - (ai + aj), we move the left pointer to the right. If it's smaller, we move the right pointer to the left. If they are equal, the problem is solved. If the pointers touch, the problem is considered unsolvable.
