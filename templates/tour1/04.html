Task D: Division by 3

Given a positive integer N. What is the minimum number of digits that need to be crossed out from N to make the remaining number divisible by 3 without a remainder?

Input Format

A single line containing one integer N (10 ≤ N ≤ 10^9).

Output Format

Output the answer to the problem.

Examples

Input:
12
Output:
0

Input:
148
Output:
1

Input:
10
Output:
1

Solution

Author: @Mazgagzam


num = input()

def solution(num):
    modulo = int(num) % 3
    if modulo == 0:
        return 0
    else:
        for char in num:
            if int(char) % 3 == modulo:
                return 1
    return 2

print(solution(num))
