Task A: Paid Parking

The cost of the first hour in the parking lot is 100 tenge. Starting from the second hour, the cost is 50 tenge per hour. How much will it cost to leave the car in the parking lot for N hours?

Input Format

A single line containing one integer N (1 ≤ N ≤ 24).

Output Format

Output the answer to the problem.

Example

Input:
3
Output:
200

Solution

Author: @Mazgagzam


n = int(input())
print(100 + (n - 1) * 50)

Is it efficient enough?

Very efficient, passes the tests easily

Complexity:

Time complexity: O(1)

Explanation:

The solution calculates the cost of parking for N hours.
