Task C: Fairy Tales

Little Kuanys was reading a collection of Kazakh fairy tales and came across the following story: Khan invited Kozhanasyr and asked him: "Kozheke, here is one som and one tiyn. Tell me, which one is more?" After thinking a little, Kozha said: "Of course, one tiyn is more. Why do you think so? Because one som has six letters. And in one tiyn there are seven letters. Well, think for yourself, which is more, six or seven?" After this story, Kuanys began to ponder such a question: "If we ask Kozhanasyr what is more, a som or b tiyn, what will he answer?" Help to find the answer to this question.

Input Format

A single line containing two integers a, b (1 ≤ a, b ≤ 9).

Output Format

When comparing like Kozhanasyr, if a som is greater, output '>', if b tiyn is greater, output '<', if they are equal, then output '='. Output without quotes.

Examples

Input:
1 1
Output:
<

Input:
9 5
Output:
>

Input:
2 3
Output:
=

Solution

Author: @Mazgagzam


nums = [0, 3, 3, 2, 4, 3, 4, 4, 5, 5]

c1, c2 = [nums[int(num)] for num in input().split(" ")]
c2 += 1
if c1 > c2:
    print(">")
elif c1 < c2:
    print("<")
else:
    print("=")

Is it efficient enough?

Very efficient, passes the tests easily

Complexity:

Time complexity: O(1)

Explanation:

The solution compares the number of letters in the Kazakh words for the given values of a and b.
