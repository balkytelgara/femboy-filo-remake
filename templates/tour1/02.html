Task B: Problems

According to Bauyrzhan, there should be 5 problems in the olympiad: 2 easy, 2 medium, and 1 hard. Currently, he has A easy, B medium, and C hard ready problems. How many olympiads can Bauyrzhan organize using only these ready problems?

Input Format

A single line containing three integers A, B, C (0 ≤ A, B, C ≤ 2 · 10^9).

Output Format

Output the answer to the problem.

Examples

Input:
5 8 3
Output:
2

Input:
100 100 0
Output:
50

Input:
520 480 178
Output:
178

Solution

Author: @Mazgagzam


ez, med, hd = [int(num) for num in input().split(" ")]
answer = min((ez // 2, med // 2, hd // 1))
print(answer)

Is it efficient enough?

Very efficient, passes the tests easily

Complexity:

Time complexity: O(1)

Explanation:

The solution calculates the number of olympiads Bauyrzhan can organize using the available ready problems.
