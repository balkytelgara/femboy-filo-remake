Task C: Houses

In a row, there are N houses, and the i-th house is located at point xi. A house is considered safe if there is another house at a distance of no more than K. Otherwise, the house is unsafe. The distance between houses i and j is given by |xi − xj |, the absolute value of the difference between xi and xj.

Output the number of houses where it is unsafe.

Input Format

In the first line, there are two integers N and K (1 ≤ N ≤ 10^5, 0 ≤ K ≤ 10^9).

In the second line, there are N integers x1, x2, · · · , xN (0 ≤ xi ≤ 10^9).

Output Format

Output the number of houses where it is unsafe.

Example

Input:
6 4
1 30 9 15 19 3
Output:
2

Solution

Author: @wilbursoot_fan


houses_amount, safe_distance = [int(num) for num in input().split(" ")]

houses = [int(num) for num in input().split(" ")]

sorted_houses = sorted(houses)

unsafe = 0
for index, house in enumerate(sorted_houses):
    if index > 0:
        if abs(house - sorted_houses[index - 1]) <= safe_distance:
            continue
    if index < houses_amount - 1:
        if abs(house - sorted_houses[index + 1]) <= safe_distance:
            continue
    unsafe += 1

print(unsafe)

Is it efficient enough?

Very efficient, passes the tests easily

Complexity:

Time complexity: O(n * log(n)) || O(houses_amount * log(houses_amount))

Explanation:

The solution sorted the houses to make the calculations easier. Then, it iterates though every house and if it has a safe neighbour uses continue to skip to the next house. If the conditions aren't met, it marks 1 house as unsafe in unsafe variable.
