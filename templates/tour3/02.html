Task B: Alice and Liana

Alice, being a very experienced programmer, decided to test Liana. Liana was given the sides of a rectangle n and m. Based on these sides, find the minimum number of squares into which it can be cut. If the answer is more than three, then output that the answer is greater than three.

Input Format

In a single line, there are two integers n and m (1 ≤ n, m ≤ 100).

Output Format

If the minimum number of squares is greater than 3, output ">3". Otherwise, output the minimum number of squares into which the rectangle n × m can be cut.

Examples

Input:
5 5
Output:
1

Input:
4 6
Output:
3

Input:
7 1
Output:
>3

Solution

Author: @wilbursoot_fan


width, height = [int(num) for num in input().split(" ")]

def alice_and_liana(width, height):
    steps = 0
    while width > 0 or height > 0:
        steps += 1
        if steps > 3:
            return ">3"
        square_size = max(min(width, height), 1)
        width = max(width - square_size, 0)
        height = max(height - square_size, 0)
    return steps

print(alice_and_liana(width, height))

Complexity:

Time complexity: O(min(max(width, height), 3)) || O(3) in worst case

Explanation:

The provided code defines a function alice_and_liana that takes the width and height of a rectangle as input and calculates the minimum number of squares needed to cover the entire rectangle. It iteratively subtracts the size of the squares from the remaining width and height until both dimensions become zero. If the number of steps exceeds three, the function returns ">3". The code then calls the function with user-input width and height values and prints the result. The algorithm efficiently determines the minimum number of squares required to cover the rectangle, emphasizing simplicity and readability.
