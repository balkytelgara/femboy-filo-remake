Task A: Room Area

Baurzhan is inside a rectangular room. He measured the distances from his position to each side of the room, resulting in values a, b, c, and d.

Determine the area of the room.

Input Format

In a single line, there are 4 integers a, b, c, d (1 ≤ a, b, c, d ≤ 100).

Output Format

Output the answer to the problem.

Example

Input:
2 3 4 5
Output:
48

Solution

Author: @wilbursoot_fan


a, b, c, d = map(int, input().split())
width = a + c
height = b + d
area = width * height
print(area)

Is it efficient enough?

Very efficient, passes the tests easily

Complexity:

Time complexity: O(1)

Explanation:

The area of the rectangle is calculated by multiplying the lengths of its sides, so the result is width * height = area, and that is the answer.
